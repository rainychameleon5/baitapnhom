thực hành mã nguồn mở
1- Thiết kế phần mềm ứng dụng giải hệ phương trình tuyến tính n phương trình n ẩn
2- Thiết kế phần mềm hỗ trợ học tập môn giải tích
3- Thiết kế phần mềm hỗ trợ học tập môn hình học
4- Xây dựng chương trình tạo báo cáo học phần môn học dựa trên dữ liệu đã cho (file csv
trên link GitHub)
5- Xây dựng phần mềm hỗ trợ môn học xử lý tín hiệu số
6- Cho file dữ liệu Student_Performance.csv. Hãy dự đoán hiệu quả học tập của sinh viên
bằng các thuật toán đã biết.
7- Thiết kế ứng dụng xác định chất lương nước dựa trên các thuật toán học máy với dữ liêu
từ file water_potability.csv
8- Thiết kế ứng dụng lọc làm mịn ảnh
9- Viết chương trình thực hiện tách biên ảnh
10- Viết chương trình tăng cường chất lượng các ảnh chụp thiếu ánh sáng
11- Dựa vào dữ liệu trong file Student_Performance.csv, hãy viết chương trình phân
tích các dữ liệu ( tìm max, min, trung bình, các đồ thị phân bố… )
12- Thiết kế game bắn chim
13- Thiết kế game nhặt hoa quả
